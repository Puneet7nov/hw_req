Hardware and OS requirements for other Server Components

Component load

The component load must be interpreted based on the following definitions. The reference CPU used in this document is:

  • Microsoft Windows or Linux servers:
      • Intel Xeon 5120, 1.87 GHz.
  • For Sun® Solaris® servers:
      • UltraSPARC® IV+, 2.10 GHz.

    The reference CPU is a unit used in this document to express:

  • The hardware requirements of each component, and
  • The total capacity of a server.

    • **The reference processors do not represent the minimum requirements for hosting ION server components.


    Load must be read as follows:

  • Very High
      • The component is highly multi-threaded and can consume multiple CPUs under peak conditions.
  • High
      • The component can use up to 100% of the reference CPU processing power under peak conditions.
  • Medium
      • The component can use up to 50% of the reference CPU processing power under peak conditions.
  • Low
      • The component can use up to 10% of the reference CPU processing power under peak conditions.

    Peak conditions are sporadic high traffic and high-volume conditions that may last for a few minutes.
